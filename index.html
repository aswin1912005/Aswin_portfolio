Projects

BINMESSAGE

BINMESSAGE is a web-based tool that allows users to encode text messages into binary and send them via WhatsApp, as well as convert binary back into readable text. It’s built using HTML, CSS, and JavaScript and features a clean, interactive interface with glowing buttons and smooth transitions.

QR CODE GENERATOR

The project aims to generate QR codes based on user input, making it easy to share information in a scannable format. Users can enter text, URLs, or other data, and the program will generate a corresponding QR code image.

BANKING SYSTEM

The Banking System in C is a simple console-based project that simulates basic banking operations like account creation, deposit, withdrawal, and balance inquiry. The program stores user data in files or structures, ensuring persistence and easy access.

AGROBOT

AgroChatbot is a web-based chatbot designed to provide instant agricultural information to farmers, students, and enthusiasts. It uses the Wikipedia API to fetch real-time data about crops, pests, farming techniques, and more.
